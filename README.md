# DK GRIDAS Marketing Report

This repository contains a client-facing HTML marketing performance report for SIA DK GRIDAS, formatted as a single static page for easy sharing and viewing in a browser.

## Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Go to Settings -> Pages.
3. Under Source, choose "Deploy from a branch".
4. Select the `main` branch and `/ (root)` folder, then Save.

Live URL format: https://<GITHUB_USERNAME>.github.io/<REPO_NAME>/

Note: If assets don't load, check relative paths.
